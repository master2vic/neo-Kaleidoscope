# neo-Kaleidoscope

个人学习llvm使用,使用c++重写官方提供的ocamlbuild实现的Kaleidoscope。

我之前学SICP的时候接触过Lisp语言和一些变种Racket。所以对这个Kaleidoscope比较容易。
